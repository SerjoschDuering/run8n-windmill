Create a new Windmill script with proper structure.

Ask user for:
1. Script name (e.g., "fetch_data")
2. Language: python | typescript | bash | go
3. Folder path within f/ (e.g., "etl", "notifications")
4. Brief description

Then create:
- `f/{folder}/{name}.{ext}` - Main script with boilerplate
- `f/{folder}/{name}.script.yaml` - Metadata with schema

Use appropriate boilerplate for the language with typed main function and example parameters.
